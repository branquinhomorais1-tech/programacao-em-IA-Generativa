📚 Sobre o Curso
Este repositório contém os registros, anotações, exercícios e materiais desenvolvidos durante o curso de Programação em Inteligência Artificial Generativa.
O objetivo do curso é explorar os principais fundamentos de IA moderna, incluindo:


Inteligência Artificial


Aprendizado de Máquina


Redes Neurais Artificiais


Processamento de Linguagem Natural (PLN)


Visão Computacional


Modelos Personalizados


Desenvolvimento com IA Generativa
